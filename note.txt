PENAMAAN FILE HTML
1. Praktek Baik Penamaan File HTML
	1.1 Jangan pernah menggunakan huruf besar, selalu gunakan huruf kecil
		contoh: index.html, home.html, about.html
	1.2 Jangan menggunakan spasi pada nama file, jika butuh pemisah gunakan - atau _
		contoh: belajar-html.html, about_page.html
	1.3 Jika butuh membuat folder gunakan aturan yang sama
